  1. Controlul conectarii la Internet pentru Updatarea automata

     1.1 Setarea URL-ului pentru verificarea ultimelor versiuni
Parametru = urlVersiuni
Valoare =http\://static.anaf.ro/static/10/Anaf/update1/versiuni.xml
Acest parametru poate fi sters sau comentat (de exemplu adaugand un "X" in fata liniei respective). In acest caz la pornirea programului DUKIntegrator nu se va mai incerca verficarea si descarcarea ultimelor versiuni de declaratii

     1.2 Setarea proxy-ului
Daca conexiunea dumneavoastra Internet se face prin intermediul unui proxy server va trebui sa adaugati linia:
proxy=auto
pentru a incerca o detectare automata a acestuia sau liniile:
proxyIP=xxx.yyy.zzz.www (adresa DNS)
proxyPort=n
pentru indicarea explicita a IP-ului sau adresei DNS/portului (prin lipsa 80) folosit de proxy server
Exemplu:
proxyIP=111.111.111.111
proxyPort=80

     1.3 Ignorarea declaratiilor noi (neinstalate)
Adaugati parametrul:
ignoraDeclaratiiNoi=D


     2. Setare tip implicit declaratie
ATENTIE ! Aceasta setare se modifica automat la fiecare inchidere a programului cu ultima valoare selectata de utilizator !
Se modifica parametrul tipDeclaratie, de exemplu:
tipDeclaratie=D300

     3. Setare tip implicit smart card
ATENTIE ! Aceasta setare se modifica automat la fiecare inchidere a programului cu ultima valoare selectata de utilizator !
Se modifica parametrul defSmartCard, de exemplu:
defSmartCard=aladdin

     4. Setare folder care se deschide dupa apoasarea butonului Browse
ATENTIE ! Aceasta setare se modifica automat la fiecare inchidere a programului cu ultima valoare selectata de utilizator !
Se modifica parametrul defFolder, de exemplu:
defFolder=C:/declaratii/xml
